# ONLINE - ACADEMY
A full-stack web learning platform built with ExpressJS + Handlebars + Supabase (PostgreSQL).
chương trình web thể hiện toàn bộ các chức năng được yêu cầu trong [file yêu cầu thực hiện](https://hackmd.io/ajKJ8RWJQoaL5Kybj_VCMw?view) project cuối kì

## Cách thức Cài đặt
1. Clone đồ án về máy
Tải file src.zip trên Google Drive của nhóm về máy, extract file, sau đó mở trong Vs Code

2. Cài đặt các thư viện cần thiết (chạy 2 dòng lệnh dưới trong terminal)

npm i express knex pg express-session passport passport-google-oauth20 bcryptjs nodemailer

npm i multer express-handlebars express-handlebars-sections sanitize-html dotenv dayjs morgan

3. Tạo file .env chứa các nội dung sau:

4. Dòng lệnh chạy đồ án
npm start

Sau khi thực hiện hết các bước trên, đồ án của nhóm đã được chạy trên đường dẫn http://localhost:3001
